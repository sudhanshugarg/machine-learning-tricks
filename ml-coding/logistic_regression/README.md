# Binary Classification with Logistic Regression

## Problem Statement

Implement logistic regression from scratch to solve a binary classification problem. Given a dataset with features and binary labels (0 or 1), train a logistic regression model and make predictions.

## Requirements

1. Implement logistic regression without using scikit-learn
2. Use gradient descent for optimization
3. Support batch predictions
4. Evaluate using accuracy, precision, recall, and F1-score

## Input

- Training data: features matrix X (n_samples, n_features) and labels y (n_samples,)
- Test data: features matrix X_test

## Output

- Trained model that can make predictions
- Probability predictions for test samples
- Model coefficients and intercept
